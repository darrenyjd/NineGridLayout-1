# NineGridLayout

## DEMO

![](http://olpu32iyy.bkt.clouddn.com/18-8-17/22405145.jpg)

## 需求

+ 满足0-9个图的适配

## 属性

```java

app:ngl_gridSpace="10dp"
app:ngl_oneChildHeight="200dp"

```

### 属性说明

+ ngl_gridSpace 表示间距
+ ngl_oneChildHeight 当只有一个图的时候的高度

|      属性值      |   说明   |
|:---------------:|:-------:|
|  ngl_gridSpace | 间距 |
|  ngl_oneChildHeight   |   当只有一个图的时候的高度 |
